Gulp Starter Kit
====

website build tools.  
use submodule add.  

## Description

* pug -> html
* styl -> css
* js(webpack&babel) -> js
* img(image-min) -> img
* other(copy) -> other

## Requirement

node.js

## Install

    git submodule add [this_repo_url] _app
    cd _app/
    node install.js

## Usage

    cd _app/
    gulp
    gulp build
    gulp watch
    gulp lint
    gulp [html|css|js|img]
    gulp [html|css|js|img]:build
    gulp [html|css|js|img]:watch
    .
    .
    .

check other tasks.

    gulp --tasks
